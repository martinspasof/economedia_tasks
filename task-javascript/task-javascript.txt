С чист JavaScript (plain js, "VanillaJS", без използването на JavaScript библиотеки) 
да се реализира следната функционалност:
- Да се извлекат статиите от основния списък (новините от деня) от http://www.dnevnik.bg/allnews/today/, 
като за всяка статия се вземе заглавие, описание, дата, автор, брой коментари, адрес на снимка към статията.

- Да се визуализират данните за статиите таблично в HTML страница.

- Да се добави към таблицата сортиране по колоните:
    -- по дата на публикуване (хронологично)
    -- по автор
    -- по брой коментари
